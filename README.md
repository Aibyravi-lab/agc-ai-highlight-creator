# 🎮 AGC - AI Gaming Highlight Creator


## Overview

AGC is an AI-powered platform that converts long gaming videos and streams into short viral highlights.


## Mission

Enable gamers and streamers to automatically generate YouTube Shorts, Instagram Reels, and TikTok clips using AI.
